# comregex 

A website application by Group 1. It checks whether a string is valid or not for the following regular expressions: 
- (a + b) (a + b)* (aa + bb) (ab + ba) (a + b)* (aba + baa)
- (11 + 00) (1 + 0)* (101 + 111 + 01) (00* + 11*) (1 + 0 + 11)
